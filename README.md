# 🚀 Crypto Price Tracker (Python & Flask)

This is my first professional Python project, developed in **Norway**. It's a real-time cryptocurrency price tracker that fetches data from the Binance API and displays it on a local network.

## 🛠 Features
* **Real-time Data:** Fetches live Bitcoin (BTC) prices.
* **Network Access:** Allows any device on the same Wi-Fi (like a smartphone) to view the price.
* **Backend:** Built using the **Flask** web framework.
* **Clean UI:** Simple and easy-to-read web interface.

## 🚀 How to Run
1.  Ensure you have Python installed on your Mac.
2.  Install the required library:
    ```bash
    pip install flask requests
    ```
3.  Run the application:
    ```bash
    python3 main.py
    ```
4.  Open the local IP address provided in the terminal on your phone or laptop.
